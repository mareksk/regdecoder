
 The 'regdecoder' tool is a simple Python application that
 each software engineer may find useful in every day work.


 REQUIREMENTS
    This app requires Python, PyQT4 and QT4 libraries.

    The app requires also one or more regs_*.xml files
    to be prepared for regular work. See 'SYNTAX.XML'
    file for more details.

 CONFIGURATION

    This app does not require an additional configuration,
    except installation of system components mentioned
    in  'REQUIREMENTS' section.
